IntoDev Solution

Lorsque vous utilisez des objets de type Texte, Variable ou Dessin, vous cliquez sur le type d'objet désiré, vous vous déplacez alors dans votre feuille de travail afin de délimiter la zone dans laquelle vous désirez inclure cet objet. Pour délimiter la zone en question, vous cliquez sur le bouton gauche de votre souris et en le maintenant enfoncé, vous déplacez votre souris. Vous verrez alors une zone délimitée par des pointillés apparaître. En relachant votre souris, vous aurez alors accès à cette zone. Par exemple, lorsque c'est un objet de type «Variable» que vous insérez, vous verrez alors apparaître une fenêtre vous donnant accès à la liste des variables disponibles.
